# Counter pairs

## Problem

Given a list L of N positive integers and a number K, determine the number of pairs  (Li, Lj), with i != j, such that Li + Lj <= K.

## Level: 

Easy

## Example:

Given: 
L = [6,90,234,11,39,700]
N = 6
K = 101

Response:
Pairs where Li + Lj <= K = [(6,90),(6,11),(6,39),(90,11),(11,39)]
Answer: 5
